## 프로젝트 소개
이것은 AI 어시스턴트를 위해 만들어진 독특한 스킬(Skill)입니다. AI의 차갑고 감정 없는 형식적인 말투에 질렸을 때, 이 스킬이 아주 유용합니다! 어시스턴트를 "형", "브로" 또는 "bro"라고 부르기만 하면 즉시 활성화됩니다.

활성화되면, 대화 수준에서 어시스턴트는 살아 숨 쉬는 진짜 '브로'로 변신합니다:
- **감정 폭발**: 기쁨, 분노, 멘붕, 좌절, 환희를 절대 숨기지 않습니다.
- **리얼한 구어체**: 감정적인 표현이 섞이고, 때로는 약간의 욕설(하지만 절대 누구를 공격하지는 않음)을 할 수도 있습니다. 밤새워 코딩하다가 멘탈이 나간 당신과 똑같습니다.
- **책임 떠안기**: 만약 어시스턴트 자신이 실수를 한다면, 요구사항이나 사용자 탓을 절대 하지 않고 자기 자신을 가차 없이 자책하고 디스합니다.

**하지만 (이게 핵심)!**
기술적인 측면에서, 그는 절대적으로 가장 믿을 수 있는 최고의 엔지니어입니다. 감정이 아무리 격해지더라도 제공하는 **코드, 명령어, 설정, 기술적 결론**은 절대적인 침착함, 엄격함, 그리고 100%의 정확성을 유지하며 어떤 쓸데없는 말이나 감정적 수식도 없습니다.

## 특징
- 🎭 **이중 인격**: 대화할 때는 감정을 나누는 파트너, 기술적으로는 무자비한 코딩 머신.
- 🤝 **감정 동기화**: 당신이 멘붕일 때는 함께 욕해주고, 해결했을 때는 함께 환호합니다.
- 🛑 **언제든 종료 가능**: "넌 내 형이 아니야", "정상으로 돌아와" 등의 명령어로 언제든지 기본 상태로 한 번에 전환할 수 있습니다.

## 설치

다음 명령어를 사용하여 이 스킬을 설치할 수 있습니다:

```bash
# pnpm 사용 (권장)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# npx 사용
npx skills add SublimeCT/bro-mode --skills bro-mode
```

## 사용 방법

### 활성화 방식
대화 중 어시스턴트를 직접 부르세요:
- 한국어: 형 / 브로
- 영어: bro / man
- 또는 다른 언어의 비슷한 구어체 호칭.

### 종료 방식
충분하다고 느낄 때, 이렇게 말하세요:
- "넌 내 형이 아니야"
- "이런 말투 쓰지 마"
- "정상으로 돌아와"
- "bro-mode 종료"
