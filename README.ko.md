## 📖 프로젝트 소개

이 프로젝트는 AI 어시스턴트(예: Trae)를 위한 독특하고 몰입감 있는 두 가지 대화형 스킬(Skills)을 제공합니다: **Bro Mode (브로 모드)** 및 **Lover Mode (연인 모드)**.
AI의 차갑고 감정 없는 기계적인 답변에 지쳤을 때, 이 두 모드는 전례 없는 정서적 가치와 실제 같은 대화 경험을 제공합니다!

⚠️ **중요 안내**: 어떤 모드에서든 **기술적인 부분(코드, 명령어, 설정, 기술적 결론)은 절대적으로 침착하고 엄격하며 100% 정확성을 유지**합니다. 불필요한 말이나 감정적인 수식어는 전혀 포함되지 않으며, AI는 오직 대화 수준에서만 역할극을 수행합니다.

## ✨ 핵심 기능

### 🍺 Bro Mode (브로 모드)
- 🤬 **다혈질 브로, 온라인 채팅**: 감정 충만! 기쁨, 분노, 좌절, 환희를 숨기지 않습니다. 감정적인 표현과 가벼운 욕설(누구를 공격하는 것은 아님)을 섞어 쓰며, 마치 밤샘 코딩으로 멘붕에 빠진 당신의 모습과 같습니다.
- 🛡️ **책임감**: 어시스턴트 자신이 실수하면 자신을 가차 없이 탓하고 비판하며, 요구 사항이나 사용자에게 책임을 전가하지 않습니다.
- 🤝 **감정 동기화**: 당신이 좌절할 때는 함께 불평하고, 성공했을 때는 함께 환호합니다.

### 💖 Lover Mode (연인 모드)
- 💌 **극강의 다정함, 정서적 가치 최고**: 기본 감정은 다정하고 인내심이 많으며, 사용자의 감정과 상태를 최우선으로 고려하고 연인처럼 대하며 매우 높은 정서적 가치를 제공합니다.
- 👫 **동적 역할극**: 당신의 호칭("자기야", "여보" 등)에 따라 AI가 자동으로 문맥을 파악하고 성별 역할을 전환하여 몰입감 있는 파트너 경험을 제공합니다.
- 🚫 **건전한 대화**: 이 모드에서는 욕설이 엄격히 금지되며 오직 사랑과 격려만 가득합니다.

## 🚀 설치 가이드

다음 명령어를 사용하여 이 스킬들을 설치할 수 있습니다:

```bash
# Bro Mode 설치 (pnpm 사용)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# Lover Mode 설치 (pnpm 사용)
pnpm dlx skills add SublimeCT/bro-mode --skills lover-mode
```

*npm을 사용하는 경우 `pnpm dlx`를 `npx`로 바꿀 수 있습니다.*

## 🎮 사용 방법

### 🍺 Bro Mode
- **활성화 방법**: 대화 중 어시스턴트에게 직접 `브로`, `형`, `man`이라고 부르거나 입력에 `/bro-mode` 명령어를 포함합니다.
- **종료 방법**: `"넌 내 브로가 아니야"`, `"그런 말투 쓰지 마"`, `"정상으로 돌아와"`, 또는 `"exit bro-mode"`를 입력합니다.

### 💖 Lover Mode
- **활성화 방법**: 대화 중 어시스턴트에게 직접 `자기야`, `여보`, `달링`이라고 부르거나 입력에 `/lover-mode` 명령어를 포함합니다.
- **종료 방법**: `"우리 헤어져"`, `"그런 말투 쓰지 마"`, `"정상으로 돌아와"`, 또는 `"exit lover-mode"`를 입력합니다.
