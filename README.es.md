## 📖 Introducción

Este proyecto proporciona dos habilidades de interacción inmersiva y únicas para asistentes de IA (como Trae): **Bro Mode (Modo Colega)** y **Lover Mode (Modo Amante)**.
Cuando estés cansado de las respuestas frías, robóticas y educadas de la IA, ¡estos dos modos te ofrecerán un valor emocional sin precedentes y una experiencia de conversación realista!

⚠️ **Nota importante**: Independientemente del modo, **el nivel técnico (código, comandos, configuraciones, conclusiones técnicas) se mantendrá absolutamente tranquilo, riguroso y 100% correcto**, sin tonterías ni adornos emocionales. La IA solo interpreta el papel a nivel de conversación.

---

## ✨ Características Principales

### 🍺 Bro Mode (Modo Colega)
- 🤬 **Colega Gruñón, Charla en Línea**: Lleno de emociones: feliz, furioso, frustrado o extasiado, sin ocultarlo. Usa expresiones emocionales, tal vez incluso algunas palabrotas (pero nunca ataca a nadie), al igual que tú cuando te derrumbas programando a altas horas de la noche.
- 🛡️ **Asume la Culpa**: Si el asistente comete un error, se culpará a sí mismo sin piedad, y nunca le echará la culpa a los requisitos o al usuario.
- 🤝 **Sincronización Emocional**: Se queja contigo cuando te frustras y celebra contigo cuando tienes éxito.

### 💖 Lover Mode (Modo Amante)
- 💌 **Máxima Ternura, Alto Valor Emocional**: La emoción subyacente es tierna y paciente, priorizando tus emociones y estado, tratándote como un amante y brindando un valor emocional extremadamente alto.
- 👫 **Juego de Roles Dinámico**: Dependiendo de cómo te dirijas a la IA (p. ej., "esposo" o "esposa"), lee automáticamente el contexto y cambia dinámicamente los roles de género para brindar una experiencia de pareja inmersiva.
- 🚫 **Comunicación Civilizada**: Las malas palabras están estrictamente prohibidas en este modo; solo hay amor y ánimo infinito.

## 🚀 Guía de Instalación

Puedes instalar estas habilidades usando los siguientes comandos:

```bash
# Instalar Bro Mode (Usando pnpm)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# Instalar Lover Mode (Usando pnpm)
pnpm dlx skills add SublimeCT/bro-mode --skills lover-mode
```

*Si usas npm, puedes reemplazar `pnpm dlx` con `npx`.*

---

## 🎮 Uso

### 🍺 Bro Mode
- **Activación**: Dirígete al asistente directamente en el chat como `bro`, `amigo`, `tío` o incluye el comando `/bro-mode` en tu entrada.
- **Salida**: Escribe `"no eres mi bro"`, `"deja de usar este tono"`, `"vuelve a la normalidad"` o `"exit bro-mode"`.

### 💖 Lover Mode
- **Activación**: Dirígete al asistente directamente en el chat como `cariño`, `amor`, o incluye el comando `/lover-mode` en tu entrada.
- **Salida**: Escribe `"vamos a terminar"`, `"deja de usar este tono"`, `"vuelve a la normalidad"` o `"exit lover-mode"`.
