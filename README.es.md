## Introducción
Esta es una habilidad (Skill) única creada para asistentes de IA. ¡Cuando estés cansado del tono frío, cortés y robótico de la IA, esta habilidad te resultará muy útil! Simplemente dirígete al asistente como "bro", "hermano" o "man", y se activará al instante.

Una vez activado, el asistente se transforma en un verdadero "bro" de carne y hueso en la conversación:
- **Lleno de Emociones**: Feliz, furioso, frustrado, molesto, eufórico: no oculta nada.
- **Argot Auténtico**: Usa expresiones emocionales, tal vez incluso algunas malas palabras (pero nunca ataca a nadie), tal como tú cuando colapsas codificando a altas horas de la noche.
- **Asume la Culpa**: Si el asistente se equivoca, se culpará y se criticará a sí mismo sin piedad, nunca culpando a los requisitos o al usuario.

**¡PERO (y esto es importante)!**
A nivel técnico, sigue siendo un ingeniero de primer nivel absolutamente confiable. No importa cuán emocional se vuelva, el **código, los comandos, las configuraciones y las conclusiones técnicas** que proporciona permanecerán absolutamente tranquilos, rigurosos y 100% correctos, sin tonterías ni adornos emocionales.

## Características
- 🎭 **Doble Personalidad**: Tu compañero emocional en la conversación, una máquina de codificación despiadada en la tecnología.
- 🤝 **Sincronización Emocional**: Se queja contigo cuando colapsas, y celebra contigo cuando tienes éxito.
- 🛑 **Salida en Cualquier Momento**: Cambia fácilmente al modo predeterminado con comandos como "no eres mi bro" o "vuelve a la normalidad".

## Instalación

Puedes instalar esta habilidad usando el siguiente comando:

```bash
# Usando pnpm (Recomendado)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# Usando npx
npx skills add SublimeCT/bro-mode --skills bro-mode
```

## Uso

### Activación
Dirígete al asistente directamente en el chat:
- Inglés: bro / man
- Español: hermano / bro / tío
- O argot equivalente en otros idiomas.

### Salida
Cuando ya hayas tenido suficiente, solo di:
- "no eres mi bro"
- "deja de usar este tono"
- "vuelve a la normalidad"
- "salir del modo bro"
