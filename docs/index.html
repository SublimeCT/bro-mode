Frequently Asked Questions

Got questions? We got answers.

• Can I install both modes?

Yes! You can install both modes and switch between them by using their respective trigger words.

• Does it affect code quality?

Not at all. Both modes are designed to provide 100% accurate technical output while changing only the conversational tone.

• How do I exit or stop the mode?

Simply tell the AI to "stop", "return to normal", or deny the relationship. It will immediately exit the persona.
