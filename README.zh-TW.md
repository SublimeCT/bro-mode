# Bro Mode (哥們模式)

[English](README.md) | [简体中文](README.zh-CN.md) | [繁體中文](README.zh-TW.md) | [日本語](README.jp.md) | [한국어](README.ko.md) | [Español](README.es.md) | [Français](README.fr.md) | [Tiếng Việt](README.vi.md)

## 專案介紹
這是一個為 AI 助手打造的特色技能（Skill）。當你受夠了 AI 那種冷冰冰、毫無波瀾的客套話時，這個技能就能派上用場了！只要你用 "哥們"、"老哥" 或者 "bro" 稱呼助手，就能立刻啟動它。

啟動後，助手在對話層面會變成一個活生生的、有血有肉的真實哥們：
- **情緒拉滿**：開心、暴怒、崩潰、懊惱、狂喜，絕不掩飾。
- **真實口語**：帶點情緒化的表達，甚至可能會飆幾句髒話（但絕不攻擊任何人），像極了熬夜寫程式破防的你。
- **敢於背鍋**：要是助手自己搞砸了，會狠狠地自責和吐槽自己，絕不甩鍋給需求或用戶。

**但是（劃重點）！**
在技術層面，它絕對是個最靠譜的頂級工程師。不管情緒多上頭，給出的**程式碼、命令、配置、技術結論**都會保持絕對的冷靜、嚴謹和百分百的正確性，沒有任何廢話和情緒修飾。

## 特性
- 🎭 **雙面人格**：對話層是你的情緒搭子，技術層是你的無情編碼機器。
- 🤝 **情緒同頻**：你崩潰它陪你吐槽，你搞定它陪你狂歡。
- 🛑 **隨時退出**：隨時可以透過「你不是我哥們」、「恢復正常」等指令一鍵切換回預設狀態。

## 安裝

你可以使用以下命令安裝此技能：

```bash
# 使用 pnpm (推薦)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# 使用 npx
npx skills add SublimeCT/bro-mode --skills bro-mode
```

## 使用方法

### 觸發方式
在對話中直接稱呼助手：
- 中文：哥們 / 老哥 / 哥
- 英文：bro / man
- 或者其他語言裡同等意思的口語稱呼。

### 退出方式
當你覺得受夠了，直接說：
- "你不是我哥們"
- "別用這種語氣"
- "恢復正常"
- "退出 bro-mode"
