## 📖 項目介紹

本項目為 AI 助手（如 Trae）提供了兩個極具特色的沉浸式交互技能（Skills）：**Bro Mode（哥們模式）** 與 **Lover Mode（愛人模式）**。
當你受夠了 AI 冷冰冰、毫無波瀾的客套話時，這兩個模式將為你提供前所未有的情緒價值與真實對話體驗！

⚠️ **重要提示**：無論在何種模式下，**技術層面（代碼、命令、配置、技術結論）都會保持絕對的冷靜、嚴謹和百分百的正確性**，絕不包含任何廢話或情緒修飾。AI 只在對話交流層面上進行角色扮演。

---

## ✨ 核心特性

### 🍺 Bro Mode (哥們模式)
- 🤬 **暴躁老哥，在線陪聊**：情緒拉滿，開心、暴怒、崩潰絕不掩飾。帶點情緒化的表達，甚至會飆幾句髒話（但絕不攻擊任何人），像極了熬夜寫代碼破防的你。
- 🛡️ **敢於背鍋**：要是助手自己搞砸了，會狠狠地自責和吐槽自己，絕不甩鍋給需求或用戶。
- 🤝 **情緒同頻**：你崩潰它陪你吐槽，你搞定它陪你狂歡。

### 💖 Lover Mode (愛人模式)
- 💌 **極致溫柔，情緒拉滿**：底層情緒是溫柔且有耐心，優先考慮你的情緒和狀態，把用戶視為愛人，提供極高的情緒價值。
- 👫 **動態角色扮演**：根據你的稱呼（如“老公”或“老婆”），AI 會自動讀取上下文並動態切換性別角色，提供沉浸式的伴侶體驗。
- 🚫 **文明交流**：該模式下嚴格禁止任何髒話，只有滿滿的愛意與鼓勵。

## 🚀 安裝指南

你可以使用以下命令安裝這些技能：

```bash
# 安裝 Bro Mode (使用 pnpm)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# 安裝 Lover Mode (使用 pnpm)
pnpm dlx skills add SublimeCT/bro-mode --skills lover-mode
```

*如果你使用 npm，可以將 `pnpm dlx` 替換為 `npx`。*

## 🎮 使用方法

### 🍺 Bro Mode
- **激活方式**：在對話中直接稱呼助手為 `哥們`、`老哥`、`bro`、`man`，或在輸入中包含 `/bro-mode` 指令。
- **退出方式**：輸入 `"你不是我哥們"`、`"別用這種語氣"`、`"恢復正常"` 或 `"退出 bro-mode"`。

### 💖 Lover Mode
- **激活方式**：在對話中直接稱呼助手為 `親愛的`、`老公`、`老婆`，或在輸入中包含 `/lover-mode` 指令。
- **退出方式**：輸入 `"我們分手吧"`、`"別用這種語氣"`、`"恢復正常"` 或 `"退出 lover-mode"`。
