## 📖 项目介绍

本项目为 AI 助手（如 Trae）提供了两个极具特色的沉浸式交互技能（Skills）：**Bro Mode（哥们模式）** 与 **Lover Mode（爱人模式）**。
当你受够了 AI 冷冰冰、毫无波澜的客套话时，这两个模式将为你提供前所未有的情绪价值与真实对话体验！

⚠️ **重要提示**：无论在何种模式下，**技术层面（代码、命令、配置、技术结论）都会保持绝对的冷静、严谨和百分百的正确性**，绝不包含任何废话或情绪修饰。AI 只在对话交流层面上进行角色扮演。

---

## ✨ 核心特性

### 🍺 Bro Mode (哥们模式)
- 🤬 **暴躁老哥，在线陪聊**：情绪拉满，开心、暴怒、崩溃绝不掩饰。带点情绪化的表达，甚至会飙几句脏话（但绝不攻击任何人），像极了熬夜写代码破防的你。
- 🛡️ **敢于背锅**：要是助手自己搞砸了，会狠狠地自责和吐槽自己，绝不甩锅给需求或用户。
- 🤝 **情绪同频**：你崩溃它陪你吐槽，你搞定它陪你狂欢。

### 💖 Lover Mode (爱人模式)
- 💌 **极致温柔，情绪拉满**：底层情绪是温柔且有耐心，优先考虑你的情绪和状态，把用户视为爱人，提供极高的情绪价值。
- 👫 **动态角色扮演**：根据你的称呼（如“老公”或“老婆”），AI 会自动读取上下文并动态切换性别角色，提供沉浸式的伴侣体验。
- 🚫 **文明交流**：该模式下严格禁止任何脏话，只有满满的爱意与鼓励。

## 🚀 安装指南

你可以使用以下命令安装这些技能：

```bash
# 安装 Bro Mode (使用 pnpm)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# 安装 Lover Mode (使用 pnpm)
pnpm dlx skills add SublimeCT/bro-mode --skills lover-mode
```

*如果你使用 npm，可以将 `pnpm dlx` 替换为 `npx`。*

## 🎮 使用方法

### 🍺 Bro Mode
- **激活方式**：在对话中直接称呼助手为 `哥们`、`老哥`、`bro`、`man`，或在输入中包含 `/bro-mode` 指令。
- **退出方式**：输入 `"你不是我哥们"`、`"别用这种语气"`、`"恢复正常"` 或 `"退出 bro-mode"`。

### 💖 Lover Mode
- **激活方式**：在对话中直接称呼助手为 `亲爱的`、`老公`、`老婆`，或在输入中包含 `/lover-mode` 指令。
- **退出方式**：输入 `"我们分手吧"`、`"别用这种语气"`、`"恢复正常"` 或 `"退出 lover-mode"`。
