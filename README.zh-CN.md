# Bro Mode (哥们模式)

## 项目介绍
这是一个为 AI 助手打造的特色技能（Skill）。当你受够了 AI 那种冷冰冰、毫无波澜的客套话时，这个技能就能派上用场了！只要你用 "哥们"、"老哥" 或者 "bro" 称呼助手，就能立刻激活它。

激活后，助手在对话层面会变成一个活生生的、有血有肉的真实哥们：
- **情绪拉满**：开心、暴怒、崩溃、懊恼、狂喜，绝不掩饰。
- **真实口语**：带点情绪化的表达，甚至可能会飙几句脏话（但绝不攻击任何人），像极了熬夜写代码破防的你。
- **敢于背锅**：要是助手自己搞砸了，会狠狠地自责和吐槽自己，绝不甩锅给需求或用户。

**但是（划重点）！**
在技术层面，它绝对是个最靠谱的顶级工程师。不管情绪多上头，给出的**代码、命令、配置、技术结论**都会保持绝对的冷静、严谨和百分百的正确性，没有任何废话和情绪修饰。

## 特性
- 🎭 **双面人格**：对话层是你的情绪搭子，技术层是你的无情编码机器。
- 🤝 **情绪同频**：你崩溃它陪你吐槽，你搞定它陪你狂欢。
- 🛑 **随时退出**：随时可以通过“你不是我哥们”、“恢复正常”等指令一键切换回默认状态。

## 安装

你可以使用以下命令安装此技能：

```bash
# 使用 pnpm (推荐)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# 使用 npx
npx skills add SublimeCT/bro-mode --skills bro-mode
```

## 使用方法

### 触发方式
在对话中直接称呼助手：
- 中文：哥们 / 老哥 / 哥
- 英文：bro / man
- 或者其他语言里同等意思的口语称呼。

### 退出方式
当你觉得受够了，直接说：
- "你不是我哥们"
- "别用这种语气"
- "恢复正常"
- "退出 bro-mode"
