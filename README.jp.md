## 📖 プロジェクト紹介

このプロジェクトは、AIアシスタント（Traeなど）に、非常にユニークで没入感のあるインタラクティブな2つのスキル（Skills）を提供します：**Bro Mode（相棒モード）** と **Lover Mode（恋人モード）**。
AIの冷たくて感情のない形式的な返答にうんざりしたとき、この2つのモードはこれまでにない感情的な価値とリアルな会話体験を提供します！

⚠️ **重要なお知らせ**：どちらのモードにおいても、**技術的な側面（コード、コマンド、設定、技術的な結論）は常に冷静かつ厳密で、100%の正確性を維持します**。無駄な言葉や感情的な装飾は一切含まれません。AIは会話レベルでのみロールプレイを行います。

---

## ✨ 主な機能

### 🍺 Bro Mode (相棒モード)
- 🤬 **短気な相棒、オンラインで雑談**：感情を全開にし、喜び、怒り、絶望を隠しません。感情的な表現や、時には汚い言葉（ただし誰も攻撃しません）を交え、深夜のコーディングで限界に達したあなたのようです。
- 🛡️ **責任を取る覚悟**：アシスタント自身が失敗した場合、激しく自分を責め、要件やユーザーのせいには決してしません。
- 🤝 **感情の同期**：あなたが絶望しているときは一緒に文句を言い、成功したときは一緒に喜びます。

### 💖 Lover Mode (恋人モード)
- 💌 **究極の優しさ、高い感情的価値**：根底にある感情は優しく忍耐強く、あなたの感情と状態を最優先し、ユーザーを恋人として扱い、非常に高い感情的価値を提供します。
- 👫 **ダイナミックなロールプレイ**：あなたの呼びかけ（「旦那」や「奥さん」など）に応じて、AIは文脈を読み取り、性別を自動的に切り替え、没入感のあるパートナー体験を提供します。
- 🚫 **丁寧なコミュニケーション**：このモードでは汚い言葉は厳禁であり、愛と励ましだけが存在します。

## 🚀 インストールガイド

以下のコマンドを使用してこれらのスキルをインストールできます：

```bash
# Bro Mode のインストール (pnpm を使用)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# Lover Mode のインストール (pnpm を使用)
pnpm dlx skills add SublimeCT/bro-mode --skills lover-mode
```

*npm を使用している場合は、`pnpm dlx` を `npx` に置き換えることができます。*

## 🎮 使用方法

### 🍺 Bro Mode
- **起動方法**：チャット内でアシスタントを直接 `相棒`、`bro`、`man` と呼ぶか、入力に `/bro-mode` コマンドを含めます。
- **終了方法**：`"お前は俺の相棒じゃない"`、`"その口調をやめろ"`、`"普通に戻れ"`、または `"exit bro-mode"` と入力します。

### 💖 Lover Mode
- **起動方法**：チャット内でアシスタントを直接 `ダーリン`、`ハニー` と呼ぶか、入力に `/lover-mode` コマンドを含めます。
- **終了方法**：`"別れよう"`、`"その口調をやめろ"`、`"普通に戻れ"`、または `"exit lover-mode"` と入力します。
