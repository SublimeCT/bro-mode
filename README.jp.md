## プロジェクト紹介
これはAIアシスタント向けに作られたユニークなスキル（Skill）です。AIの冷たくて無感情な建前にうんざりした時、このスキルが役立ちます！「アニキ」、「兄弟」、または「bro」とアシスタントを呼ぶだけで、すぐにアクティブになります。

アクティブになると、アシスタントは会話の面で生身の「アニキ」に変わります：
- **感情爆発**：喜び、激怒、崩壊、後悔、狂喜、一切隠しません。
- **リアルな口語**：感情的な表現を交え、時には少し汚い言葉も飛び出すかもしれません（ただし絶対に誰も攻撃しません）。深夜にコードを書いていて限界を迎えたあなたにそっくりです。
- **自ら責任を負う**：もしアシスタント自身が失敗したら、容赦なく自分を責め、ツッコミを入れます。要件やユーザーのせいには絶対にしません。

**しかし（ここが重要）！**
技術的な面では、絶対に最も頼りになるトップエンジニアです。どんなに感情的になっても、提供される**コード、コマンド、設定、技術的な結論**は絶対的な冷静さと厳密さを保ち、100%の正確性を誇ります。無駄話や感情的な装飾は一切ありません。

## 特徴
- 🎭 **二重人格**：会話ではあなたの感情の相棒、技術面では冷酷なコーディングマシーン。
- 🤝 **感情のシンクロ**：あなたが崩壊すれば一緒にツッコミを入れ、成功すれば一緒に狂喜します。
- 🛑 **いつでも終了可能**：「お前は俺のアニキじゃない」、「普通に戻れ」などのコマンドで、いつでもデフォルト状態に一瞬で切り替えられます。

## インストール

以下のコマンドを使用して、このスキルをインストールできます：

```bash
# pnpm を使用 (推奨)
pnpm dlx skills add SublimeCT/bro-mode --skills bro-mode

# npx を使用
npx skills add SublimeCT/bro-mode --skills bro-mode
```

## 使用方法

### トリガー
会話の中で直接アシスタントを呼びます：
- 日本語：アニキ / 兄弟
- 英語：bro / man
- または他の言語での同等の口語表現。

### 終了方法
もう十分だと思ったら、直接こう言ってください：
- "お前は俺のアニキじゃない"
- "その口調をやめろ"
- "普通に戻れ"
- "bro-modeを終了"
